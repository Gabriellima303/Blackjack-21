# 🃏 Blackjack em Java

Projeto desenvolvido em Java.

## 🎮 Funcionalidades

* Jogo de Blackjack funcional
* Sistema de compra de cartas (Hit)
* Parar jogada (Stand)
* Crupiê automático
* Carta do crupiê escondida durante o jogo


## ▶️ Como executar

### Opção 1: Rodar o .jar

```bash
java -jar blackjack.jar
```

### Opção 2: Rodar pelo código

Abra o projeto em uma IDE e execute a classe:

```
BlackjackGUI.java
```
